איך הופכים את Claude Desktop ל-RTL (תמיכה מלאה בעברית) 🇮🇱

מאת עידן גינדי עודכן יולי 2026 זמן קריאה: 6 דקות · זמן ביצוע: כ-5 דקות

אפליקציית Claude Desktop למק מציגה את הכל משמאל-לימין, מה שהופך שיחות בעברית ללא נוחות. במדריך הזה נהפוך את כל הממשק ל-RTL בעזרת פאץ' קטן של CSS. לוקח כ-5 דקות, וקל לביטול.

לפני שמתחילים: זה שינוי לא-רשמי בקבצי האפליקציה. הוא בטוח והפיך (יוצרים גיבוי אוטומטי), אבל ייתכן שתצטרכו להריץ אותו שוב אחרי כל עדכון של Claude.

מה צריך

  • מק עם Claude Desktop מותקן (בתיקיית Applications).
  • Node.js מותקן. לבדיקה: פותחים Terminal ומריצים node -v. אם חוזר מספר גרסה - יש לכם. אם לא - מורידים מ-nodejs.org.

שלב 1 - יוצרים את קובץ ה-CSS

צרו קובץ בשם claude-rtl.css בתיקיית ההורדות (Downloads) עם התוכן הבא:

/* Claude Desktop — RTL patch */

html, body, #root, #app,
[class*="App"], [class*="app_"] {
  direction: rtl !important;
}

p, div, span, li, ul, ol,
h1, h2, h3, h4, h5, h6,
label, button, a {
  text-align: start;
}

[contenteditable="true"], textarea,
input[type="text"], input[type="search"],
.ProseMirror,
[class*="message"], [class*="Message"], [class*="prose"] {
  unicode-bidi: plaintext !important;
  text-align: start !important;
}

/* קוד, קישורים וטקסט אנגלי נשארים LTR */
code, pre, kbd, samp, tt,
[dir="ltr"],
[class*="code"], [class*="Code"], [class*="hljs"] {
  direction: ltr !important;
  text-align: left !important;
  unicode-bidi: plaintext;
}

שלב 2 - יוצרים את סקריפט ההתקנה

באותה תיקייה, צרו קובץ בשם apply-rtl-patch.sh עם התוכן הבא:

#!/usr/bin/env bash
set -euo pipefail

APP="/Applications/Claude.app"
RES="$APP/Contents/Resources"
ASAR="$RES/app.asar"
BACKUP="$RES/app.asar.backup"
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
CSS="$SCRIPT_DIR/claude-rtl.css"

# מצב שחזור: bash apply-rtl-patch.sh --restore
if [[ "${1:-}" == "--restore" ]]; then
  if [[ -f "$BACKUP" ]]; then
    cp "$BACKUP" "$ASAR"; echo "✅ הגרסה המקורית שוחזרה. פתחו מחדש את Claude."
  else
    echo "⚠️  לא נמצא גיבוי — אין מה לשחזר."
  fi
  exit 0
fi

[[ -d "$APP" ]] || { echo "❌ Claude.app לא נמצא"; exit 1; }
[[ -f "$CSS" ]] || { echo "❌ claude-rtl.css לא נמצא ליד הסקריפט"; exit 1; }
command -v npx >/dev/null 2>&1 || { echo "❌ צריך Node.js — nodejs.org"; exit 1; }

echo "→ מגבה את app.asar..."
[[ -f "$BACKUP" ]] || cp "$ASAR" "$BACKUP"

WORK="$(mktemp -d)"
echo "→ מחלץ..."
npx --yes @electron/asar extract "$BACKUP" "$WORK"

echo "→ מזריק את ה-CSS..."
STYLE="<style id=\"claude-rtl-patch\">$(cat "$CSS")</style>"
INJECTED=0
while IFS= read -r -d '' html; do
  grep -q "claude-rtl-patch" "$html" && continue
  if grep -q "</head>" "$html"; then
    python3 - "$html" "$STYLE" <<'PY'
import sys
path, style = sys.argv[1], sys.argv[2]
html = open(path, encoding="utf-8").read().replace("</head>", style + "</head>", 1)
open(path, "w", encoding="utf-8").write(html)
PY
    INJECTED=$((INJECTED+1))
  fi
done < <(find "$WORK" -name "*.html" -print0)

echo "→ הוזרק ל-$INJECTED קבצים."
echo "→ אורז מחדש..."
npx --yes @electron/asar pack "$WORK" "$ASAR"
rm -rf "$WORK"
echo "✅ סיום! סגרו לגמרי את Claude (Cmd+Q) ופתחו מחדש."

שלב 3 - מריצים

  1. סגרו את Claude לגמרי - Cmd+Q (לא רק את החלון).
  2. פותחים Terminal.
  3. מדביקים את השורה הבאה ולוחצים Enter:
cd ~/Downloads && bash apply-rtl-patch.sh

מחכים להודעה ✅ סיום, ואז פותחים מחדש את Claude.

זהו - הממשק אמור להיות עכשיו מימין-לשמאל. 🎉

איך מבטלים

אם משהו לא מוצא חן בעיניכם, מריצים:

cd ~/Downloads && bash apply-rtl-patch.sh --restore

זה מחזיר את הגרסה המקורית מהגיבוי.

פתרון תקלות

  • "command not found: node" - Node.js לא מותקן. הורידו מ-nodejs.org.
  • "No such file or directory" - שני הקבצים לא נמצאים בתיקייה שאליה נכנסתם. ודאו ששניהם ב-~/Downloads והריצו ls *.sh *.css כדי לבדוק.
  • אחרי עדכון של Claude חזר להיות LTR - זה נורמלי. פשוט מריצים את הסקריפט שוב.
רוצים לפתוח בלחיצה כפולה במקום Terminal? אפשר להפוך את הסקריפט לקובץ .command, אבל macOS יחסום אותו כ"מפתח לא מזוהה". במקרה כזה: קליק ימני ‹ Open, ואם אין כפתור Open - System Settings ‹ Privacy & Security ‹ "Open Anyway".

הפאץ' משנה רק CSS בתוך האפליקציה ולא נוגע בנתונים או בהגדרות. יוצר גיבוי אוטומטי ותמיד ניתן לשחזור.

לא בא לכם לגעת ב-Terminal בעצמכם?

יש גם דרך לגרום ל-Claude עצמו להכין לכם את שני הקבצים האלה ולתת פקודת הרצה מותאמת למחשב שלכם - ראו פרומפטים מוכנים להעתקה. יש גם פאצ'ים קהילתיים מוכנים מראש (לא מהאתר הזה) אם אתם מעדיפים לא לגעת בקוד בכלל - חפשו "claude-desktop-rtl-patch" בגיטהאב.

נתקעתם באמצע, או שיש אלמנט שלא מתיישר נכון אחרי הפאץ'? אפשר לדבר.
